# Star Wars List
#### By Ash Laidlaw

This is a simple React application that pulls a list of Star Wars characters from https://swapi.co/ and allows the deletion of those characters from the list. The goal was to learn about how to implement components and pass data back and forth between them.

### Technologies Used
* JavaScript
* HTML
* CSS / Bootstrap
* React
* Webpack

### Instructions to Use

Clone the Repo, and run `npm install`. Then, run `npm build start` to start the app.
